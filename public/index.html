How it works

From playlist to progress.

CourseTube keeps the workflow simple: bring a playlist, turn it into a course, then study lesson by lesson with the right tools beside the video.

1

Paste a YouTube playlist

Add a public playlist link and choose the pace that matches how deeply you want to study.

2

Generate a course structure

The app organizes videos into modules, lessons, checkpoints, practice tasks, and a course objective.

3

Study inside one workspace

Watch the video, read the summary, write notes, paste transcripts, and run focus sessions without leaving the course.

Learning workspace

Everything a playlist needs to become usable.

Embedded lessons Play public YouTube videos directly in the course window.
Summary tab Generate a concise lesson overview, key points, and a question to answer.
Notes tab Keep course notes saved with your learning path.
Transcript tab Paste a transcript manually when YouTube does not expose readable captions.
Focus timer Start study or break sessions while staying in the lesson.
Progress tracking Mark lessons complete and resume later from your dashboard.

YouTube integration

For playback, public YouTube videos are embedded directly in the course window.

For reliable playlist import, set `YOUTUBE_API_KEY` before starting the server. Without it, the app uses the current no-key importer and manual fallback.
